.forEach() Array.prototype.forEach

描述：对 Array 的各元素执行一个函数

参数：
fn:Function每个元素需要执行的函数
thisObj:Object传递内部函数执行时this对象

举例：

//doms 是一个数组 Array of Elements
var temp = [],doms = [HtmlDivElement,HtmlHeadingElement,HtmlSpanElement,HtmlAnchorElement];
doms.forEach(function(dom, index, array){
	if(NL.utils.isInArray(dom.tagName.toLowerCase(), ['h1', 'div', 'p']))
		temp.push(dom);
});//temp = [HtmlDivElement, HtmlHeadingElement]
