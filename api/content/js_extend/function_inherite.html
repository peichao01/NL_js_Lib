.inherite() Function.prototype.inherite

描述：继承自其它构造器

参数：
Class:Function需要继承的父类构造函数

注：很显然，只有类的 实例方法 能被继承。

返回值：

举例：

function Animal() { }
Animal.method('run',function () {
    alert('用脚奔跑');
});

function Person(name) {
    this.name = name;
}
//一定要在添加原型方法之前继承，否则，原型方法会被抹掉
Person.inherite(Animal);

Person.addProto({
    say: function () {
        alert("I'm " + this.name);
    }
});

var p = new Person('John');
p.say();//输出：I'm John
p.run(); //输出：用脚奔跑
