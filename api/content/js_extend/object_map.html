Object.map() Object.prototype.map

描述：对Object的各元素执行一个函数，函数的返回值替换原元素

参数：
object:Object
fn:Function每个元素需要执行的函数
[ thisObj:Object ]传递内部函数执行时this对象

注：

返回值：返回一个新的对象

举例：

var o = { name:'JOhN',sex:'MaLe'};
o = Object.map(o, function(el, index, array){
	return el.toLowerCase();
})//o = { name:'john',sex:'male'}
