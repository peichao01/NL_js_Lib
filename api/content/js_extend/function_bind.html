.bind() Function.prototype.bind

描述：为函数绑定 this 对象 ||version Mozilla 实现版

参数：
thisObj :Object 需要绑定的 函数的 this指向

返回值：一个新的函数

举例：

var o = {
	sayThis: function () {
		alert(this);
	}
}
o.sayThis();//输出：[object Object]--o
var say = o.sayThis;
say();//输出：[object DOMWindow]--window
var say2 = say.bind(o);
say2();//输出：[object Object]--o
