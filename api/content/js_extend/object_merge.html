Object.merge() Object.merge

描述：合并对象，把source的属性都传递给接受的对象并返回

参数：
destination:Object接受属性的对象
source:Object输出属性的对象
[ isOverride:Boolean ]接受者已经有了同名属性时，是否覆盖

注：

返回值：

举例：

var User = {
    Main: function (name) {
        this.name = name;
    },
    say: function () {
        alert(this.name + ' said: Hello!');
    }
}

Object.merge(User, {age:20});

var bob = Object.construct(User,'John');
alert(bob.age);//输出：20
