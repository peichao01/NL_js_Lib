Object.construct() Object.construct

描述：创建一个新的对象，把参数作为原型，且，若参数原型中含有 Main 函数，即先初始化 再返回

参数：
proto :Object新对象的原型，且先初始化（若有 Main 函数）
[ Main_para:Mixin ]Main 函数 所需要的参数

注：

返回值：new Object

举例：

var User = {
    Main: function (name) {
        this.name = name;
    },
    say: function () {
        alert(this.name + ' said: Hello!');
    }
}

var bob = Object.construct(User,'John');
bob.say();//输出： John said: Hello
