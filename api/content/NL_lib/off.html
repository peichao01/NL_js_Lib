NL.off()

描述：移除事件的方法( DOM && non-DOM )

参数：
ID:Array
[ removeAll:Boolean ]是否全部移除，只适用于 non-DOM 事件

注：类似PHP的文件操作,fopen()返回一个资源句柄，fclose()函数把句柄传进来即可

返回值：

举例：

NL.off(ID);

var ID1 = NL.on('fuck', function (voice, v2, v3) {
    alert(voice);
    NL.off(ID1);//执行一次，即删除
}, 'n~~~', 'aaa~~~');

var ID2 = NL('#id').on('click', function(e){
    NL.off(ID2);
})
