NL.on()

描述：non-DOM 的监听事件的方法

参数：
eventType:String监听的事件类型
callback:Function回调函数
[ fnParams:Mixin ]回调函数的参数

注：on 后面可以跟任意多个 fire 调用时使用的 参数

返回值：ID: Array

举例：

NL.on('customEvent', fn);
NL.on({ 'customEvent': fn, 'loaded': fn2 })
