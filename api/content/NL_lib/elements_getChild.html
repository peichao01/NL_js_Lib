NL.elements.getChild()

描述：获取 dom 元素的第 index 个子元素

参数：
dom:Element
index:Number获取元素的索引
filters:Object参见 nodeListFilter
[ notCareTextNode:Boolean ]忽略文本节点

注：内部方法，请使用 NL('selector').getChild()

返回值：child : Element

举例：

<div id="a">aaaa<p></p><span>span01</span><b></b><span class="cls"></span></div>

NL.console.log(NL.elements.getChild((NL('#a').doms)[0], 0, { tagName: 'span' }, true));
//输出: <span>span01</span>
